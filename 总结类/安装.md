# Linux
你可以从包管理器中安装Zig。   

大多数主要的linux发行版已经支持了最新的Zig版本。  

举个例子，如果你的系统是Debian系列(比如Ubuntu)，运行下面的命令：  
```bash
sudo apt-get install zig
```

# MacOS

直接在命令行运行如下命令：

```bash
brew install zig
```
